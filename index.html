Runa Cat Ears

Summary

Installation

  1. Unpack the dependencies, in order.
  2. Unpack Runa_Cat_Ears.unitypackage.
  3. Drag Assets/Kemono_Io/Runa_Cat_Ears.prefab into the Runa_Source object.
  4. Constrain the Runa_Cat_Ears' armature to the head.
    1. Select the Runa_Cat_Ears's armature in the Hierarchy panel.
    2. Drag the head bone into the Parent Constraint's Source in the Inspector panel.
    3. Click the Activate button.
  5. Set the Runa_Cat_Ears's anchor override.
    1. Select the Runa_Cat_Ears mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  6. Add the ears to the emission and hue shift animations.
    1. Select the Runa_Source object in the Hierarchy panel.
    2. Set the Animator's Controller to FX_Runa in the Inspector panel.
    3. Select the Animation tab and click the lock icon if it is not already locked.
    4. Select the Runa_Cat_Ears mesh.
    5. Select the color_0 animation and click the record button.
    6. Set the Hue Shift properties to 0.
    7. Select the color_1 animation and click the record button.
    8. Set both Hue Shift properties to 1.
    9. Select the body_lightning_0 animation and click the record button.
    10. Set the Emission Strength property to 0.1.
    11. Select the body_lightning_1 animation and click the record button.
    12. Set the Emission Strength property to 1.
    13. Click the Preview button to exit record mode.
    14. Click the lock icon.
    15. Right click the Animator's Controller and select Revert.
